# Random-Music-Picker
Petit outil permettant de sélectionner aléatoirement une musique à écouter à partir d’une liste prédéfinie. En cliquant sur un bouton, une recommandation musicale apparaît. Ce projet m’a permis de pratiquer la logique JavaScript, la génération aléatoire et la manipulation du DOM.  ⚙️ Technologies : HTML • CSS • JavaScript
